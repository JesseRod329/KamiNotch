# Contributing to KamiNotch

Thanks for your interest in KamiNotch.

## How to Contribute
- Open an issue to discuss a bug, feature, or design change before starting work.
- Keep changes small and focused.
- Include clear repro steps for bugs.
- For features, describe the user impact and any UI changes.

## Development Setup
Project setup instructions will be added once the app scaffolding is in place.

## Pull Requests
- Link the issue your PR addresses.
- Keep PRs scoped to a single change.
- Describe testing you performed.

## Code of Conduct
By participating, you agree to the Code of Conduct in `CODE_OF_CONDUCT.md`.
